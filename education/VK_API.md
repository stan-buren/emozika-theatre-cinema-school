# API и Токены (CS50 Bridge)

В CS50 мы говорили, что программы могут общаться друг с другом. **API (Application Programming Interface)** — это как меню в ресторане.  
Вы (ваша программа) не идете на кухню (сервер ВКонтакте) готовить бургер. Вы говорите официанту (API): "Принеси мне бургер (данные)".

## Что такое Токен?
В CS50, когда вы сдавали задачи, система проверяла, что это именно вы.  
**Service Token (Сервисный ключ)** — это ваш бейдж сотрудника. Он говорит ВКонтакте: *"Привет, это сайт Эмоцики. Я свой, дай мне данные группы"*.

---

## 🚀 Практика: Как получить VK Service Token

Чтобы наш скрипт мог "читать" стену группы, нам нужен этот ключ.

### Шаг 1. Создать Приложение
1.  Зайдите на [dev.vk.com/admin](https://dev.vk.com/admin).
2.  Нажмите кнопку **"Создать приложение"**.
3.  Название: `Emozika Website` (или любое понятное вам).
4.  Платформа: **Сайт** (или Standalone/Встраиваемое — для сервис-токена подойдет Standalone).
    *   *Совет: выберите "Standalone-приложение" (оно самое гибкое для скриптов).*

### Шаг 2. Получить Токен
1.  В меню слева выберите **Настройки**.
2.  Найдите поле **"Сервисный ключ доступа"** (Service Token).
3.  Нажмите иконку копирования. Это и есть ваш `VK_SERVICE_TOKEN`.

> ⚠️ **Важно:** Никому не показывайте этот ключ. Если кто-то его украдет, он сможет делать запросы от имени вашего приложения.

### Шаг 3. Узнать ID Группы
1.  Зайдите в вашу группу ВКонтакте.
2.  Посмотрите на URL.
    *   Если там `vk.com/club123456`, то ваш ID — `123456` (число).
    *   Если там буквенный адрес (например, `vk.com/teatr.emocika`), откройте любое фото в группе. Ссылка будет вида `photo-58293658_457242...`.
    *   Первое число после `photo-` (и обычно с минусом) — это ID. В нашем случае: `58293658`.

### Шаг 4. Вставить в проект
Откройте файл `.env` в корне проекта и запишите:

```env
VK_SERVICE_TOKEN=ваш_длинный_ключ_от_ВК
VK_GROUP_ID=-58293658
```
*(Обратите внимание: для скриптов API мы часто используем минус перед ID группы, если нас просят `owner_id`. Но наш скрипт умеет обрабатывать и просто число).*
