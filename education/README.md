# 🎓 База Знаний (Education Hub)

Привет, коллега! 👋
Раз ты прошел **CS50**, ты уже знаешь базу: алгоритмы, память, C, Python, SQL, HTML/CSS.
Этот раздел — твой мост от CS50 к современному фронтенду.

Здесь я объясняю инструменты, которые мы используем в проекте, простым языком, проводя аналогии с тем, что ты уже учил.
