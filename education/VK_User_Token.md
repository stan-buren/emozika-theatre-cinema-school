# Как получить VK User Token (Простой способ)

Мы используем публичный ID популярного приложения (Kate Mobile), чтобы не создавать своё. Это безопасно и работает сразу.

### 1. Нажмите на эту ссылку
[👉 ПОЛУЧИТЬ ТОКЕН (Нажмите здесь)](https://oauth.vk.com/authorize?client_id=2685278&scope=video,offline&response_type=token&v=5.199&display=page&redirect_uri=https://oauth.vk.com/blank.html)

### 2. Разрешите доступ
Нажмите кнопку **"Разрешить"** (Allow).

### 3. Скопируйте Токен
1.  После нажатия вы попадете на белую страницу.
2.  Посмотрите в **адресную строку браузера** (сверху).
3.  Там будет длинная строка:
    `https://oauth.vk.com/blank.html#access_token=vk1.a.ВОТ_ЭТОТ_ДЛИННЫЙ_ТЕКСТ...&expires_in=...`
4.  Скопируйте всё, что после `access_token=` и до `&`.

### 4. Вставьте в проект
В файл `.env`:
```bash
VK_SERVICE_TOKEN=vk1.a.ВашСкопированныйТокен
```

### 4. Копируем Токен
1.  В адресной строке вы увидите что-то типа:
    `https://oauth.vk.com/blank.html#access_token=vk1.a.ДЛИННЫЙ_НАБОР_БУКВ...&expires_in=0...`
2.  Скопируйте всё, что находится между `access_token=` и `&expires_in`.
3.  Это и есть ваш **User Token**.

### 5. Сохраняем
Вставьте этот токен в файл `.env` вашего проекта:
```bash
VK_SERVICE_TOKEN=vk1.a.ВашНовыйТокен...
```
*(Мы используем эту переменную, скрипт поймет)*
